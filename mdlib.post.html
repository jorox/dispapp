mdlib.post.disptools.sort_eigvals(qsord: numpy.array, eigenqs: numpy.array) → numpy.array¶

Sort eigenvalues by distance from previous point

The function tries to sort the eigenvalues for each q_i vector by finding the closest eigenvalue to the previous e_j to e_i-1. The algorithm does not always produce correct results, but is fast

Note

Assumes that the q-vectors have to be in increasing order

Note

don’t use this. Just sort eigenvalues by increasing

Parameters

  • qsord (np.array) – nx3 array of eigenvectors in order

  • eigenqs (np.array) – nx3 array of eigenvalues

Returns

return an eigenvalue array now sorted

Return type

np.array

mdlib.post.disptools.sort_eigvals_lsq(qsord: numpy.array, eigenqs: numpy.array) → numpy.array¶

Sort eigenvalues by sum of least-squares method

Calculate the distance between (q_i,e_i,j) and (q_i+1,e_i+1,k) for all permutations of k, and the find the permutation that produces the least sum of squares of the distances j-k.

Parameters

  • qsord (np.array) – nx3 array of eigenvectors in order

  • eigenqs (np.array) – nx3 array of eigenvalues

Returns

return an eigenvalue array now sorted

Return type

np.array

Note

Assumes that the q vectors are in inceasing order
